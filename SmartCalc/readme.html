SmartCalc V1.0

SmartCalc - это универсальное приложение-калькулятор, предлагающее три основные функции:

1. Простой Калькулятор

Выполняйте арифметические операции, находите значения функций. Если в строке калькулятора есть _x_, то после нажатия кнопки "=" предусмотрен
ввод значения переменной _x_ в диалоговом окне для подстановки значения.

2. Построение Графиков Функций

Кнопка "GRAPH" открывает график функций. Можно менять границы значений _x_ и _y_ и с помощью кнопки RESIZE менять область видимости.

3. Калькулятор Кредита

Во вкладке Credit находится кредитный калькулятор для расчета ежемесячного платежа, платежи за проценты, а также общую выплату по кредиту. При выборе дифференцированного типа платежа появляется ComboBox с номерами месяцев. Так можно посмотреть, сколько рублей в каком месяце будет составлять месячный платеж по кредиту по кредиту.

Установка и удаление

Установка приложения на рабочий стол происходит с помощью команды make install. Удалить приложение с рабочего стола можно с помощью команды make uninstall
